# java_imageManager
一个用java实现的图片浏览程序。

###功能列表：
####窗体功能：
* 目录树（点击时动态生成）  
* 目录下的缩略图展示（图片等比缩放）  
* 动态在面板中输出图片  
* 缩略图对于鼠标操作(进入，出去，点击图片，点击面板)的样式变化  
* 双击打开查看图片  
* 改变查看单张图片窗体的大小图片等比缩放  
* 窗体菜单  
* 右键图片弹出菜单  
* 右键面板弹出菜单
